# Pinnacle Foods

This is a repository for tracking the APIs, SDKs, and other developer resources for Pinnacle Foods. Pinnacle Foods was an American producer of branded packaged foods (Birds Eye, Duncan Hines, Vlasic, Mrs. Paul's, Hungry-Man) before being acquired by Conagra Brands in 2018.
